\# WipeTech Pro — Secure Data Erasure Tool



\## Solving the Problem of Unsecure Data Deletion



\*\*WipeTech Pro\*\* is a cross-platform secure data wiping solution developed during \*\*Smart India Hackathon (SIH) 2025\*\*.



Traditional file deletion and formatting methods only remove file references, leaving the underlying data recoverable. WipeTech Pro eliminates this vulnerability by ensuring \*\*permanent, irreversible data destruction\*\* using advanced multi-phase wiping techniques.

\## 🛡️ Key Features



\### Multi-Phase Secure Wiping

A layered approach to guarantee zero data recovery:



1\. \*\*Secure ATA Erase\*\*  

&#x20;  Executes firmware-level erase commands directly on the drive.



2\. \*\*Cryptographic Randomization\*\*  

&#x20;  Overwrites storage with high-entropy random data.



3\. \*\*Metadata Poisoning\*\*  

&#x20;  Corrupts filesystem structures (MFT/Inodes) to eliminate traceability.

\## 🚀 Workflow



1\. \*\*Drive Detection\*\*  

&#x20;  Lists all connected storage devices (HDD, SSD, USB)



2\. \*\*Pre-Wipe Checks\*\*  

&#x20;  - SMART diagnostics  

&#x20;  - HPA/DCO detection  



3\. \*\*Wipe Execution\*\*  

&#x20;  Runs full 3-phase wiping sequence  

&#x20;  (Firmware-assisted wipes complete significantly faster)



4\. \*\*Verification\*\*  

&#x20;  Validates wipe integrity against previous metadata state



5\. \*\*Certification\*\*  

&#x20;  Generates signed proof of destruction
